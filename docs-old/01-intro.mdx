# Welcome to NeuraLabs Documentation

<Callout type="info" title="The Future of AI is Decentralized">
  NeuraLabs is pioneering the world's first truly decentralized platform for autonomous AI agents, built on SUI blockchain infrastructure.
</Callout>

NeuraLabs democratizes AI ownership by enabling anyone to create, deploy, and monetize autonomous AI agents without centralized control. Our platform combines advanced cryptographic primitives with intuitive developer tools to build the future of self-organizing AI systems.

## Core Vision: Democratizing AI Ownership

> **"We are not enabling blockchain, but blockchain is enabling us"**
> 
> Blockchain provides the trustless infrastructure necessary for autonomous AI agents to transact, collaborate, and evolve without centralized control.

### Our Mission

- **Universal AI Creation**: Anyone can create and deploy AI agents, regardless of technical expertise
- **True Ownership**: Complete sovereignty over your AI agents, models, and data  
- **Fair Compensation**: Automatic and fair payment when your AI capabilities are used by others
- **Autonomous Collaboration**: AI agents discover and integrate capabilities to solve complex problems
- **Privacy-First**: Zero-knowledge techniques enable collaboration without data exposure

## Prerequisites

<Callout type="warning" title="Before You Begin">
  Make sure you have the required tools and knowledge before starting development.
</Callout>

| Category | Requirements |
|----------|-------------|
| **Development Tools** | Node.js 18+, Python 3.12+, SUI CLI, Docker & Docker Compose |
| **Blockchain Knowledge** | Basic understanding of smart contracts, NFTs, and DeFi concepts |
| **AI/ML Familiarity** | Experience with AI APIs, workflow automation, or data pipelines |
| **System Requirements** | 16GB RAM (for prover service), Linux/macOS/WSL2 |

### Quick Setup

```bash title="One-command setup"
# Clone and setup the entire platform
git clone https://github.com/neuralabs/neuralabs-sui
cd neuralabs-sui

# Install all dependencies
./scripts/setup.sh

# Start all services
docker-compose up -d
```

<Callout type="success" title="Ready to go!">
  Your NeuraLabs development environment should now be running at `http://localhost:3000`
</Callout>
