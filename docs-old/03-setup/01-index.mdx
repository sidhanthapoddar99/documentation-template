## Implementation Order

For a successful deployment, we recommend following this order:

1. **[Database Setup](./01-database/setup)** - Foundation for data storage
2. **[Backend Services](./03-backend/setup)** - Core API functionality
3. **[SUI Integration](/docs/sui-integration/overview)** - Complete blockchain integration
4. **[Frontend Application](./02-frontend/setup)** - User interface
5. **[HPC Execution Node](./04-hpc-execution/setup)** - AI workflow processing

## Quick Start

```bash
# Clone the repository
git clone https://github.com/neuralabs/neuralabs-sui
cd neuralabs-sui

# Start all services with Docker
docker-compose up -d

# Or follow individual setup guides for manual installation
```
