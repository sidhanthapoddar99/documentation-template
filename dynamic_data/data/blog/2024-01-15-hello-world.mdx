# Hello World

Welcome to the documentation framework blog! We're excited to share this new tool with you.

## What's New

This documentation framework offers:

- **Fast Performance** - Built on Astro for optimal speed
- **Easy Configuration** - Simple YAML configuration files
- **Beautiful Design** - Clean, modern theme with dark mode

## Getting Started

To get started with the framework:

1. Clone the repository
2. Install dependencies with `bun install`
3. Start development with `bun run dev`
