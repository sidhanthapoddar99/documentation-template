# Project Roadmap

This section outlines the development roadmap organized into 4 phases, plus a backlog for bugs and feature ideas.

---

## Development Phases

### [Phase 1: Foundation & Core Polish](./01_sprints/01_phase-1)
Essential fixes and improvements to stabilize the framework.
- Fix sizing consistency
- Code block enhancements
- Outline/TOC polish
- Blog testing
- Theme refinements

### [Phase 2: Content Features & Layouts](./01_sprints/02_phase-2)
Add rich content features and expand layout options.
- Mermaid & GraphViz diagrams
- Components library
- Additional doc layouts
- Custom page templates
- Navbar & footer variations
- Claude skills for development
- Dev-only content (hide in production)
- Multiple data paths support
- Codebase refactoring

### [Phase 3: Configuration & Deployment](./01_sprints/03_phase-3)
Production-ready deployment and mobile compatibility.
- Deployment module (Docker, Vercel, Netlify)
- Configuration enhancements
- Dev toolbar improvements
- Mobile compatibility & responsive design
- Built-in themes

### [Phase 4: Advanced Features](./01_sprints/04_phase-4)
Extensibility through plugins and AI-powered features.
- Plugin architecture
- Search implementation
- AI-powered features
- Graph view
- Interactive hydration
