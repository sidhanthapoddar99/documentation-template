# Welcome

This is the sample blog post that ships with the template. Posts live as flat
markdown files under `data/blog/` named `YYYY-MM-DD-<slug>.md`. The date in the
filename and the `date:` frontmatter should match.

## Add a post

Create a new file alongside this one:

```
data/blog/2026-05-01-my-first-real-post.md
```

…with at minimum `title`, `description`, and `date` in the frontmatter. The blog
index at `/blog` updates automatically.
