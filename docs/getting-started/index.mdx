# Getting Started

Welcome to the documentation framework! This guide will help you get up and running quickly.

## Prerequisites

Before you begin, make sure you have the following installed:

- Node.js 18 or higher
- npm or pnpm

## Installation

1. Clone the repository:

```bash
git clone https://github.com/your-org/your-repo.git
cd your-repo/astro
```

2. Install dependencies:

```bash
npm install
```

3. Configure environment:

```bash
cp .env.example .env
```

4. Start the development server:

```bash
npm run dev
```

## Project Structure

The documentation content lives outside the Astro application for easy management:

```
/project-root/
├── docs/           # Your documentation content
│   ├── getting-started/
│   ├── guides/
│   └── api/
└── astro/          # Astro application
    ├── src/
    └── ...
```
