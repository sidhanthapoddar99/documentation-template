# Configuration Overview

Welcome to the Astro Documentation Framework configuration guide. This section covers everything you need to know to customize your documentation site.

## Quick Start

1. **Site Configuration**: Define your site structure in `src/config/site.config.ts`
2. **Themes**: Customize colors and typography in `src/theme/`
3. **Layouts**: Choose or create page layouts in `src/layouts/`
4. **Components**: Use built-in MDX components or create your own

## Configuration Files

| File | Purpose |
|------|---------|
| `src/config/site.config.ts` | **Main configuration** - site structure, pages, navbar, footer |
| `src/config/types.ts` | TypeScript type definitions |
| `src/config/helpers.ts` | Helper functions for accessing config |
| `astro.config.mjs` | Astro framework configuration |
| `src/theme/*.css` | Color and typography customization |

## Site Configuration

The framework uses a centralized TypeScript configuration file for all site settings:

```typescript
// src/config/site.config.ts
export const siteConfig: SiteConfig = {
  site: {
    name: 'My Docs',
    title: 'Documentation',
    description: 'My documentation site',
  },
  defaults: {
    navbar: 'minimal',
    sidebar: 'default',
    footer: 'default',
    outline: 'default',
  },
  pages: {
    home: { type: 'home', title: 'Home', url: '/' },
    docs: { type: 'doc', title: 'Docs', url: '/docs', data_location: '../docs' },
  },
  navbar: ['home', 'docs'],
  footer: { copyright: '2024 Your Company' },
};
```
