# Frontmatter Reference

Frontmatter is the YAML metadata block at the top of every documentation file. It controls how pages appear in the sidebar, their SEO properties, and various display options.

## Overview

This section covers all available frontmatter options:

- [Basic Syntax](./basic-syntax) - How to write frontmatter
- [Required Fields](./required-fields) - The `title` field
- [SEO & Metadata](./seo-metadata) - Description, keywords, images
- [Sidebar Configuration](./sidebar-config) - Position and labels
- [Content Organization](./content-organization) - Tags and categories
- [Publishing Control](./publishing-control) - Drafts and dates
- [Page Behavior](./page-behavior) - Display options
- [Examples](./examples) - Complete and minimal examples
- [Validation](./validation) - Error handling and tips

## Quick Example

```markdown
---
title: My Page
description: A brief description
sidebar_position: 1
---

# My Page

Content goes here...
```
