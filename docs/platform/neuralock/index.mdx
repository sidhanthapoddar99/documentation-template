# Neuralock: Distributed Threshold Encryption

<Callout type="info">
  Neuralock provides secure, decentralized key management by combining Shamir's Secret Sharing with blockchain-based access control, eliminating single points of failure in encryption systems.
</Callout>

## What is Neuralock?

Neuralock is a distributed threshold encryption system that revolutionizes how applications handle encrypted data. Instead of relying on centralized key servers or complex key management infrastructure, Neuralock distributes encryption keys across multiple servers using Shamir's Secret Sharing, with permissions enforced through smart contracts.

## How It Works

<div style={{ marginBottom: '2rem' }}>
  <Card>
    <CardHeader>
      <CardTitle>1. Encryption Process</CardTitle>
    </CardHeader>
    <ol style={{ paddingLeft: '1.5rem' }}>
      <li>Client generates a random AES encryption key</li>
      <li>Data is encrypted locally using AES-256-GCM</li>
      <li>AES key is split into shares using Shamir's Secret Sharing</li>
      <li>Each share is encrypted and sent to a different server</li>
      <li>Servers verify permissions via smart contracts before storing</li>
    </ol>
  </Card>
</div>

<div style={{ marginBottom: '2rem' }}>
  <Card>
    <CardHeader>
      <CardTitle>2. Decryption Process</CardTitle>
    </CardHeader>
    <ol style={{ paddingLeft: '1.5rem' }}>
      <li>Client requests shares from multiple servers</li>
      <li>Each server checks blockchain permissions independently</li>
      <li>Minimum threshold of servers return their encrypted shares</li>
      <li>Client reconstructs the AES key from the shares</li>
      <li>Data is decrypted locally with the reconstructed key</li>
    </ol>
  </Card>
</div>

## Quick Example

<CodeBlock language="typescript">
{`// Initialize Neuralock client
const neuralock = new NeuralockClient({
  applicationContract: "0x...",
  signer: wallet,
  servers: [
    { nftId: 1 },
    { nftId: 2 },
    { nftId: 3 }
  ]
});

// Encrypt sensitive data
const encrypted = await neuralock.encrypt(
  "Secret document content",
  "doc-123"
);

// Decrypt when needed
const decrypted = await neuralock.decrypt(
  encrypted,
  "doc-123"
);`}
