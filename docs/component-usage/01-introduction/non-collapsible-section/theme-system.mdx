# Theme System

The NeuraLabs documentation uses a sophisticated theme system for consistent styling.

## Color Management

Our color system is centralized in `src/theme/colors.js`:

- **Brand Colors**: Primary, secondary, accent colors
- **Semantic Colors**: Success, warning, error, info
- **Theme Variables**: Automatically adapt to light/dark mode

## Using Theme Variables

```css
.custom-component {
  background: var(--theme-bg-surface);
  color: var(--theme-text-primary);
  border: 1px solid var(--theme-border-subtle);
}
```

## Generating Colors

After modifying `colors.js`:

```bash
node src/theme/generateColors.js
```

This generates the CSS variables used throughout the site.
