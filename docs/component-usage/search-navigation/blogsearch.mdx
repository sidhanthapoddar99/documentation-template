# BlogSearch Component

The BlogSearch component provides real-time search functionality for blog posts with highlighting and filtering capabilities.

## Import

```javascript
import { BlogSearch } from '@site/src/components/elements/BlogSearch';
```

## Basic Usage

```jsx
import { BlogSearch } from '@site/src/components/elements/BlogSearch';

function BlogPage() {
  return (
    <div className="blog-header">
      <h1>Blog</h1>
      <BlogSearch />
    </div>
  );
}
```

## Features

### Real-time Search
- Searches as you type (after 3 characters)
- No need to press Enter
- Instant results display

### Search Scope
The component searches across:
- Post titles
- Post descriptions
- Post tags

### Results Display
- Shows up to 8 results
- Displays post title, description preview, date, and tags
- Highlights matching text
- Click to navigate to post

### Search Highlighting
Matching terms are highlighted in the results:

```jsx
<mark className="search-highlight">matched text</mark>
```

## Behavior

### Search Trigger
- Minimum 3 characters to trigger search
- Debounced to avoid excessive filtering

### Navigation
- Click on result to navigate to post
- Search box clears after navigation
- Results dropdown closes

### No Results
When no posts match the search:
```jsx
<div className="blog-search-no-results">
  <p>No blog posts found for "{searchTerm}"</p>
  <small>Try different keywords or check your spelling</small>
</div>
```

## Implementation Details

### Using Docusaurus Hooks
```javascript
import { useBlogPosts } from '@docusaurus/theme-common/internal';

const blogPosts = useBlogPosts();
```

### Filtering Logic
```javascript
const filtered = blogPosts.filter(post => {
  const title = post.metadata.title.toLowerCase();
  const description = post.metadata.description?.toLowerCase() || '';
  const tags = post.metadata.tags.map(tag => tag.label.toLowerCase()).join(' ');
  const searchLower = searchTerm.toLowerCase();
  
  return title.includes(searchLower) || 
         description.includes(searchLower) || 
         tags.includes(searchLower);
});
```

### Text Highlighting
```javascript
const highlightText = (text, term) => {
  if (!term) return text;
  const regex = new RegExp(`(${term})`, 'gi');
  const parts = text.split(regex);
  return parts.map((part, index) => 
    regex.test(part) ? <mark key={index} className="search-highlight">{part}</mark> : part
  );
};
```

## Customization

### Placeholder Text
```jsx
<BlogSearch placeholder="Find articles..." />
```

### Results Limit
Modify the slice value to change max results:
```javascript
setFilteredPosts(filtered.slice(0, 10)); // Show 10 results
```

### Search Threshold
Change minimum characters for search:
```javascript
if (searchTerm.length > 1) { // Search after 2 characters
  // perform search
}
```

## Performance Considerations

1. **Debouncing**: Consider adding debounce for large blog collections
2. **Memoization**: Cache filtered results if performance is an issue
3. **Lazy Loading**: Load blog metadata on demand for very large sites
