

# Standard Dirs


The standard Docusaurus structure includes:
- `docs/` - Documentation pages
- `blog/` - Blog posts  
- `src/pages/` - Custom pages
- `src/components/` - React components
- `src/theme/` - Theme overrides
- `static/` - Static assets


# Common theme overrides
- `src/theme/DocItem/TOCInline.js` - Customize the Table of Contents inline behavior
- `src/theme/DocItem/Content.js` - Customize the content rendering of documentation pages
- `src/theme/BlogPostItem.js` - Customize the blog post item rendering
- `src/theme/Footer/Logo.js` - Customize the footer logo
- `src/theme/Navbar/Logo.js` - Customize the navbar logo
- `src/theme/Navbar/Content.js` - Customize the navbar content
- `src/theme/MDXComponents.js` - Customize MDX components globally
- `src/theme/NotFound.js` - Customize the 404 page
- `src/theme/Prism.js` - Customize code block syntax highlighting
- `src/theme/Head.js` - Customize the head section of HTML pages
- `src/theme/AnnouncementBar.js` - Customize the announcement bar
- `src/theme/DocSidebar/Sidebar.js` - Customize the sidebar rendering
- `src/theme/DocSidebar/Item.js` - Customize individual sidebar items
- `src/theme/DocSidebar/Category.js` - Customize sidebar categories
- `src/theme/DocSidebar/CategoryLink.js` - Customize sidebar category links
- `src/theme/DocSidebar/CategoryCollapsible.js` - Customize collapsible sidebar categories
- `src/theme/DocSidebar/CategoryLink.js` - Customize sidebar category links
