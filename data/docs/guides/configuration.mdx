# Configuration Guide

This guide covers all configuration options available in the framework.

## Site Configuration

The main site configuration is in `config/site.yaml`:

```yaml
site:
  name: "My Docs"
  title: "My Documentation"
  description: "Site description for SEO"
  logo:
    src: "/logo.svg"
    alt: "Logo"

pages:
  docs:
    base_url: "/docs"
    type: docs
    layout: "@docs/doc_style1"
    data: "@data/docs"
```

### Site Metadata

| Field | Description |
|-------|-------------|
| `name` | Site name used in navbar |
| `title` | Default page title |
| `description` | Site description for SEO |
| `logo` | Logo configuration |

### Page Definitions

Each page definition includes:

- `base_url` - URL prefix for the page
- `type` - Page type (docs, blog, custom)
- `layout` - Layout reference using @ prefix
- `data` - Data source using @ prefix

## Navbar Configuration

Configure navigation in `config/navbar.yaml`:

```yaml
layout: "@navbar/style1"

items:
  - label: "Home"
    href: "/"
  - label: "Docs"
    page: "docs"  # References page name from site.yaml
  - label: "GitHub"
    href: "https://github.com/user/repo"
    external: true
```

## Environment Variables

Configure paths in `.env`:

```env
CONFIG_DIR=./config
DATA_DIR=./data
THEMES_DIR=./themes
ENABLE_DARK_MODE=true
```
