# Getting Started

Welcome to the documentation framework! This guide will help you get up and running.

## Prerequisites

Before you begin, make sure you have:

- Node.js 18+ or Bun
- A code editor

## Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/user/repo.git
cd repo
bun install
```

## Development

Start the development server:

```bash
bun run dev
```

Open [http://localhost:4321](http://localhost:4321) in your browser.

## Project Structure

The project follows this structure:

- `config/` - Configuration files (site.yaml, navbar.yaml, footer.yaml)
- `data/` - Your content (docs, blog, pages)
- `src/` - Source code and layouts
- `themes/` - Custom themes (optional)

## Configuration

Edit `config/site.yaml` to configure your site:

```yaml
site:
  name: "My Docs"
  title: "My Documentation"
  description: "Modern documentation"
```
