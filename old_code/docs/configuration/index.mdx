# Configuration

Welcome to the configuration guide. This section covers everything you need to customize your documentation site.

## Configuration Structure

```
Configuration
├── Site Configuration     # Main config file (site.config.ts)
├── Themes                 # Colors, typography, dark mode
├── Navigation/            # Navbar and footer
│   ├── Navbar            # Navigation bar variants
│   └── Footer            # Site footer
├── Layouts/               # Page layouts
│   └── Layout Details    # DocsLayout, HomeLayout, etc.
├── Components/            # MDX components
│   └── Component Reference  # Cards, Callouts, Code blocks
└── Modules/               # Optional features
    └── Module Reference   # Search, Analytics, AI
```

## Quick Start

1. **Site Configuration** - Define your site in `src/config/site.config.ts`
2. **Themes** - Customize colors in `src/theme/`
3. **Navigation** - Configure navbar and footer
4. **Layouts** - Choose page layouts per section
5. **Modules** - Enable search, analytics, etc.

## Sections

### Site Configuration

The central configuration file that controls your entire site structure, pages, and navigation.

```typescript
// src/config/site.config.ts
export const siteConfig = {
  site: { name: 'My Docs', title: 'Documentation' },
  pages: { home: { type: 'home', url: '/' } },
  navbar: ['home', 'docs'],
  footer: { copyright: '2024' },
};
```
